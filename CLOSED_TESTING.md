# Closed testing

I'm doing [closed testing](https://play.google.com/console/about/closed-testing/) of this app, before I release it to the store, and that needs 12 testers to test it for 14 days.

If you would like to give it a go before I can release it properly:

* First, join the [braindelay-guitartools-testing](https://groups.google.com/g/braindelay-guitartools-testing) google group. The members of this list are private to me, so nobody else will see it, and nobody can send emails to you (and I never will). I'll delete this list as soon as testing is complete.
* Then use the [com.braindelay.guitartools](https://play.google.com/apps/testing/com.braindelay.guitartools) testing link to download the app.

You can leave the test at any time.
